\newline

\begin{enumerate}

\item Determinar si cada uno de los siguientes enunciados es
Verdadero o Falso. En cada caso justificar su respuesta.
\begin{enumerate}
\item Todo número entero es un número racional.

\item $0$ es a la vez un número racional y un número irracional.

\item Para todo número real $x$ las expresiones: " El triple de
$x$, más 5" y " El triple de $x$ más 5" son iguales.

\item El recíproco de $\frac{1}{a}$ es $a$.

\item Si $a$ y $b$ son números reales tales que $ab=0$ entonces
$a=0$ y $b=0$. \item El opuesto del opuesto de $-\sqrt{7}$ es
$\sqrt{7}$. \item Si $y \neq 0$ y $w \neq 0$, entonces
$\frac{x}{y} + \frac{z}{w} = \frac{x+z}{y+w}$.

\item Para todo número real $a$ se tiene que
$\frac{a+3}{5a+15}=\frac{1}{5}$.

\item Si $z \neq 0$ entonces $\frac{x+y}{z} = \frac{x}{z} +
\frac{y}{z}$.

\item El conjunto $\{1,0,-1\}$ es cerrado para la adición y para
la multiplicación.

\item Para todo número real $a \neq 0$ se tiene que $a^2 > 0$.

\item Si $a$ y $b$ son números reales tales que $a^2 = b^2$
entonces $a=b$. \item Hay más números enteros que números
naturales.
\end{enumerate}

\item Determinar si cada uno de los siguientes enunciados es
Verdadero o Falso. En cada caso justificar su respuesta.
\begin{enumerate}

\item El $5\%$ del $10\%$ de un número es el $2\%$ del número.

\item La suma de dos números positivos y un número negativo es un
número positivo.

\item Para todo número real $x$ se tiene que $-(-x)=(-1)(-x)$.

\item Para todo número real $x$, el número $-x$ es negativo.

\item Si en un producto de números hay más factores negativos que
positivos, el resultado es un número negativo.

\item Si $a,b$ y $z$ son números reales tales que $a<b$ y $-c > 0$
entonces $bc<ac$.

\item Si $x$ y $y$ son positivos y $x>y$ entonces $\frac{1}{x} >
\frac{1}{y}$.

\item Si $\frac{1}{x} \leq \frac{x}{x+1}$ entonces $x+1 \leq x^2$.

\item Si $- \frac{1}{2} < x < \frac{1}{2}$ entonces $-\frac{3}{2}
< x < \frac{3}{2}$.

\item Si $x$ y $y$ son números reales tales que $x < y$ entonces
$x^2 < y^2$.

\item Si $x,y,z$ y $w$ son números reales tales que $x < y $ y $z
< w$, entonces $x-z < y -w$.

\item Si $x$ y $y$ son números reales tales que $x \geq 0$ y $y <
0$ entonces $xy^{-1} \geq 0$.
\end{enumerate}

\item Determinar si cada uno de los siguientes enunciados es
Verdadero o Falso. En cada caso justificar su respuesta.
\begin{enumerate}
\item $1$ es un divisor de $0$ y $0$ es un divisor de $1$.

\item Si $ab$ es un entero par entonces $a$ y $b$ son enteros
pares.

\item Si $x$ es un número irracional, entonces $\sqrt{x}$ es un
número irracional.

\item Si $x,y$ y $z$ son enteros tales que $x | yz$, entonces $x |
y$ o $x | z$.

\item El menor múltiplo de 12 mayor que $-300$ es $-260$.

\item Todo entero impar se puede escribir en la forma $2n + 7$
para algún $n$ entero.

\item Si $x$ es un entero impar, entonces $x^2 - 1$ es divisible
por $8$.

\item El conjunto de los números irracionales es cerrado para la
adición y la multiplicación.

\item El inverso de un número irracional es un número irracional.

\item Existen enteros $x$ y $y$ con $y \neq 0$, tales que $x - y =
175$, el cociente obtenido al dividir $x$ por $y$ es  $15$ y el
residuo $7$.

\item La forma decimal de un número racional se obtiene dividiendo
el numerador por el denominador y se conoce completamente  cuando
se repite el primer residuo.
\end{enumerate}

\item Determinar si cada uno de los siguientes enunciados es
Verdadero o Falso. En cada caso justificar su respuesta.
\begin{enumerate}

\item $\sqrt{5}(\sqrt{2}- 1)$  es un número irracional.

\item Existen infinitos números racionales entre $0.\overline{9}$
y $1$.

\item Si $| \ x - 7 \ | \ = 2$ entonces $x = 9$.

\item Para todo número real $x$ se tiene que $| \ 2x+5 \ | \ = 2 \
| \ x \ | + 5$.

\item Si $x < -1$ entonces $| \ x \ |+| \ x+1 \ |+| \ x-2 \
|=3x+1$ .

\item Existe un entero que al dividirlo por $3$ deja residuo $2$ y
al dividirlo por $5$ deja residuo $3$.

\item Si $q$ no es divisible por $3$, el número racional positivo
$\frac{p}{q}$ tiene una expansión decimal finita.

\item El número racional positivo $\frac{p}{q}$ tiene una
expansión decimal finita si y solo si los únicos divisores primos
de $q$ son $2$ y $5$.

\item $\left| \sqrt{3} - 2 \right| + | \ -3 \ | - | \ 4 - \sqrt{3}
\ | = 2 \sqrt{3} - 9$.

\item Existen expresiones decimales infinitas no periódicas que
representan números racionales.

\item La expresión decimal $5.331333133331333331\ldots$ representa
un número irracional. \item Si $-5 \leq x + 1 \leq 3$ entonces $|
\ x + 1 \ | \leq 3$.
\end{enumerate}

\item
\begin{enumerate}
\item Encuentre $3$ números racionales y $3$ números irracionales
entre $3.14592$ y $\pi$. (Nota: $\pi = 3.145926\ldots$).

\item Cuántos números irracionales hay entre $\frac{1}{100}$ y
$\frac{1}{99}$.

\item Escribir los números racionales $5.\overline{2}$,
$23.\overline{14}$ y $2.5\overline{32}$ en la forma $\frac{p}{q}$.

\item Qué significado tiene cada una de las siguientes expresiones
$\frac{0}{1}$, $\frac{1}{0}$ y $\frac{0}{0}$?

\item Ordenar de mayor a menor cada uno de los siguientes números
y representarlos sobre la recta numérica.
\begin{center}
\begin{tabular}{ccccc}
$-2.71$ & $\sqrt{3}$ & $- \frac{2}{5}$ & $\pi$ & $21 \div
10^3$ \\
$0.\overline{2}$ & $-3.13$ & $0.0002\times 10^2$ & $-\pi$ &
$0.2\times 10^3$
\end{tabular}
\end{center}

\item Descomponer en producto de primos y hallar todos los
divisores de $2340$, $26208$ y $45747$.

\item Hallar todos los valores de $a$ y $b$ tales que $3^a2^b$
divida a $3780$ y todos los valores de $a$ tales que $2^a 3^4 5^a
11$ divida a $5940$.

\item Dar un ejemplo de un subconjunto de $\mathbf{R}$ que:
\begin{enumerate}

\item No tenga cotas superiores ni cotas inferiores.

\item Tenga cotas superiores pero no tenga cotas inferiores.

\item Tenga $Inf$ pero no tenga $Sup$.

\item No tenga ni $Inf$ ni $Sup$.
\end{enumerate}
\end{enumerate}

\end{enumerate}
