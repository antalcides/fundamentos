Representación de un polinomio en Matlab

Sea el polinomio p(x) = 5x^3 - 7x^2 - 3x + 2. En Matlab se representa mediante un vector con los coeficientes del polinomio en orden descendente:

p = [5, -7, -3, 2];

Evaluación de un polinomio en punto o conjunto de puntos, x

res = polyval(p,2.4)
x = 1:3
px = polyval(p,x)
%
x = 0:0.1:3;
y = polyval(p,x);
plot(x,y)

res =

   23.6000


x =

     1     2     3


px =

    -3     8    65

Raices de un polinomio

Soluciones de la ecuación p(x)=0

r = roots(p)

r =

    1.6180
   -0.6180
    0.4000

Si se conocen las raices de un polinomio, se puede determinar el polinomio:

pr = poly(r)

pr =

    1.0000   -1.4000   -0.6000    0.4000

Ajuste polinómico (1)

Dados N puntos (x_i , y_i) se puede obtener un polinomio de order N-1 que pasa por todos los puntos. En Matlab este polinomio se puede obtener usando el comando polyfit(x,y,N-1):

x = linspace(0,10,5);
y = rand(1,5);
p = polyfit(x,y,4);
xx = linspace(0,10,101);
z = polyval(p,xx);
plot(xx,z,x,y,'ro')

Ajuste polinómico (2)

También se puede ajustar un polinomio a un conjunto de N puntos (x_i,y_i) sin necesidad de que el polinomio pase por todos los puntos:

x = 1:7;
y = [1.9,4.4,5.8,8.6,9.6,12.8,13.5]
p = polyfit(x,y,1);
xx = linspace(1,7);
z = polyval(p,xx);
plot(xx,z,x,y,'ro')

y =

  Columns 1 through 5

    1.9000    4.4000    5.8000    8.6000    9.6000

  Columns 6 through 7

   12.8000   13.5000

En este caso, el polinomio (que tiene orden menor que (N-1) ), se calcula por el método de "Mínimos cuadrados".
